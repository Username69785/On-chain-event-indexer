# On-Chain Event Indexer

## Описание
Этот проект представляет собой индексатор событий блокчейна Solana, написанный на Rust. Он предназначен для отслеживания, сбора и сохранения информации о транзакциях и передачах токенов для заданных адресов.

## Функциональность
*   **Сбор подписей (Signatures):** Получает историю транзакций для указанного адреса, используя API (например, Helius).
*   **Фильтрация и сохранение:** Сохраняет уникальные подписи в базу данных PostgreSQL и отслеживает статус их обработки.
*   **Индексация транзакций:** Загружает полные данные транзакций для необработанных подписей.
*   **Парсинг событий:** Извлекает детали транзакций, включая:
    *   Общую информацию (слот, время, комиссии, ошибки).
    *   Детальные данные о переводах токенов (Token Transfers).
*   **Хранение данных:** Структурированное хранение данных в реляционной базе данных (PostgreSQL).

## Технологический стек
*   **Язык:** Rust
*   **Асинхронность:** Tokio
*   **HTTP Клиент:** Reqwest
*   **База данных:** PostgreSQL (через SQLx)
*   **Блокчейн:** Solana SDK / Client
*   **Логирование:** Tracing
*   **Конфигурация:** Dotenv / Config

## Структура базы данных
Основные таблицы (подробнее в `DataBaseInfo.md`):
1.  **`signatures`**: Хранит подписи транзакций и флаг `is_processed`.
2.  **`transactions`**: Общая информация о транзакции (комиссия, слот, статус).
3.  **`token_transfers`**: Детали переводов токенов (отправитель, получатель, сумма, минт).
